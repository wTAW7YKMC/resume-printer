# 简历留言后端 - JSON文件存储版本

## 项目概述

本项目是一个基于Java的简历网站留言后端服务，使用本地JSON文件存储留言数据，无需任何数据库依赖。

## 已完成功能

1. ✅ 创建基于JSON文件存储的留言后端代码
2. ✅ 实现POST提交留言接口
3. ✅ 实现GET查看留言列表接口
4. ✅ 创建JAR包生成工具
5. ✅ 生成自包含的JAR包
6. ✅ 更新run.bat和运行说明文档

## 技术实现

### 核心技术
- Java 8+
- 内置HTTP服务器 (com.sun.net.httpserver)
- Gson库用于JSON数据处理
- 读写锁 (ReentrantReadWriteLock) 保证并发访问安全

### 数据存储
- 留言数据存储在本地JSON文件 (messages.json)
- 支持并发读写操作
- 自动创建数据文件

### API接口
- POST /api/resume/message - 提交留言
- GET /api/resume/message/list - 获取留言列表
- GET / - API说明页面

## 使用方法

1. 双击run.bat启动服务
2. 服务启动后监听8080端口
3. 通过API接口提交和查看留言

## 优势

1. **零依赖** - 只需要Java环境，无需数据库或其他依赖
2. **零配置** - 开箱即用，无需任何配置
3. **自包含** - 所有依赖已打包到单个JAR文件中
4. **简单易用** - 双击即可启动，API接口清晰

## 与SQL Server版本对比

| 特性 | JSON文件存储版 | SQL Server版 |
|------|---------------|-------------|
| 依赖 | 仅需Java环境 | 需要Java + SQL Server |
| 配置 | 零配置 | 需要配置数据库连接 |
| 数据存储 | 本地JSON文件 | SQL Server数据库 |
| 性能 | 适合小规模应用 | 适合大规模应用 |
| 可靠性 | 依赖文件系统 | 数据库事务保证 |
| 部署复杂度 | 极低 | 中等 |
