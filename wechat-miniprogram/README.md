# Becky的个人简历小程序

这是一个基于微信小程序开发的个人简历展示应用，具有现代化的UI设计和流畅的用户体验。

## 功能特点

- 📱 响应式设计，适配各种屏幕尺寸
- 🎨 现代化UI设计，卡片式布局
- ⚡ 流畅的页面切换和动画效果
- 📝 打字机效果展示个人介绍
- 📊 技能进度条动态展示
- 📞 一键拨打电话和发送邮件
- 🔄 支持分享功能

## 页面结构

- **首页**: 个人信息概览和快速导航
- **关于我**: 详细个人介绍和打字机效果
- **工作经历**: 工作经验展示，支持展开/收起
- **教育背景**: 学历和教育经历
- **技能专长**: 技能水平可视化展示
- **项目经验**: 项目案例展示
- **联系方式**: 联系方式和社交媒体链接

## 开发环境

- 微信开发者工具
- 小程序基础库版本: 2.19.4

## 使用方法

1. 使用微信开发者工具打开项目
2. 在project.config.json中配置您的AppID
3. 点击编译运行项目
4. 在模拟器或真机上预览效果

## 项目结构

```
wechat-miniprogram/
├── app.js                 # 小程序入口文件
├── app.json              # 小程序全局配置
├── app.wxss              # 全局样式文件
├── pages/                # 页面目录
│   ├── index/            # 首页
│   ├── about/            # 关于我
│   ├── experience/       # 工作经历
│   ├── education/        # 教育背景
│   ├── skills/           # 技能专长
│   ├── projects/         # 项目经验
│   └── contact/          # 联系方式
├── components/           # 自定义组件
├── utils/                # 工具函数
├── data/                 # 数据文件
└── assets/               # 静态资源
```

## 自定义配置

1. 修改 `data/resume.js` 中的数据来自定义您的个人信息
2. 在 `app.json` 中配置页面路由和导航栏样式
3. 在 `app.wxss` 中调整全局样式

## 技术栈

- 微信小程序原生开发
- JavaScript ES6+
- WXSS样式
- Flexbox布局

## 注意事项

- 请确保所有图片资源已正确放置在assets目录下
- 如需使用真实数据，请替换data/resume.js中的示例数据
- 部署前请务必填写正确的AppID
